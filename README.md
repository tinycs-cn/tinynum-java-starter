# TinyNum — Java

从零实现一个教学版 NumPy（Java 版）。

## 项目结构

```
src/main/java/dev/tensorhero/tinynum/
├── NDArray.java   ← 核心实现文件
├── DType.java     ← 数据类型
└── Slice.java     ← 切片定义
tests/
├── TestE01.java ... TestE15.java
```

## 快速开始

```bash
# 编译
mvn compile

# 运行测试（需配合 tinynum-tester）
```

在 `NDArray.java` 中找到 `TODO` 注释，按 stage 顺序逐步实现。
